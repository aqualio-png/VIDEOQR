Etape A

GitHub Pages \u2014 Heberger VideoQR.html (pour le mode PIN)

Gratuit, permanent. Le scan du QR ouvre la page dans Chrome au lieu de la telecharger. Les videos restent sur votre Freebox.

1
Allez sur github.com, connectez-vous. Cliquez sur + en haut a droite \u2192 New repository. Nommez-le videoqr, cochez Public, cliquez Create repository.
2
Dans votre nouveau depot, cliquez sur Add file \u2192 Upload files. Glissez ce fichier VideoQR.html dans la zone, puis cliquez Commit changes.
3
Cliquez sur Settings (onglet en haut du depot) \u2192 dans le menu gauche cliquez Pages \u2192 sous Branch selectionnez main \u2192 cliquez Save.
4
Attendez 1 a 2 minutes. Revenez sur Settings \u2192 Pages. Vous verrez une bandeau vert avec le message "Your site is live at" suivi de votre URL. C'est ici que se trouve le lien a copier.

L'URL ressemble exactement a :
https://VOTRE-PSEUDO.github.io/videoqr/VideoQR.html

Remplacez VOTRE-PSEUDO par votre nom d'utilisateur GitHub (visible en haut a droite de github.com apres connexion).
5
Copiez cette URL. Dans VideoQR, cochez Proteger par code PIN \u2192 collez l'URL dans le champ URL de VideoQR.html. Generez votre QR. Le scan ouvrira desormais la page correctement dans Chrome.
📌 Ou trouver l'URL GitHub Pages
1. github.com \u2192 votre depot videoqr
2. Onglet Settings \u2192 menu gauche \u2192 Pages
3. Bandeau vert : "Your site is live at https://..."
4. Ajoutez /VideoQR.html a la fin si absent

Exemple final :
https://aqualio.github.io/videoqr/VideoQR.html
Etape B

GitHub Releases \u2014 Heberger les videos (optionnel)

Si vous ne voulez pas utiliser la Freebox pour les videos. Gratuit jusqu'a 2 Go par fichier, URL permanente.

1
Dans votre depot GitHub (le meme videoqr ou un autre), cliquez sur Releases dans la colonne de droite \u2192 Create a new release.
2
Dans Choose a tag, tapez v1.0 \u2192 cliquez Create new tag. Glissez votre fichier video dans la zone Attach binaries by dropping them here.
3
Cliquez Publish release. Attendez la fin de l'upload (barre de progression verte).
4
Sur la page de la release, repérez la section Assets tout en bas. Appuyez longtemps sur le nom du fichier video (Android) ou clic droit \u2192 Copier l'adresse du lien (PC).

✗ Ne pas copier depuis la barre d'adresse du navigateur \u2014 cette URL est temporaire.
✓ Copier via clic droit sur le nom du fichier dans Assets.

L'URL correcte ressemble a :
https://github.com/aqualio/videoqr/releases/download/v1.0/video.mp4
⚠ URLs a ne pas utiliser
✗ https://github.com/aqualio \u2014 profil, pas une video
✗ https://github.com/aqualio/videoqr \u2014 page du depot
✗ https://release-assets.githubusercontent.com/... \u2014 URL temporaire JWT, expire !
